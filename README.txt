TESTE. COLOQUEI O NOME PADRÃO DE Htmlreport. PARA EXECUTRAR O TESTE COM A GERAÇÃO
DO RELATÓRIO DEVE-SE INSERIR O COMANDO:

py.test -s -vc acessando_site.py --html=Htmlreport.html
